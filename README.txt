Instrucciones para probar el programa en heroku: Ingrese en el navegador la url https://program6.herokuapp.com/calculation?p=<valor>&dof=<valor>
												Ejemplo: https://program6.herokuapp.com/calculation?p=0.20&dof=6
Instrucciones para compilar el codigo fuente: Ejecute el comando mvn package desde psp2.1-program-5
Instrucciones para desplegar la aplicación localmente:  Ejecute el comando heroku local web desde psp2.1-program-6
Instrucciones para ejecutar el programa: Ingrese en el navegador la url localhost:5000/calculation?p=<valor>&dof=<valor>
										 Ejemplo: localhost:5000/calculation?p=0.20&dof=6
